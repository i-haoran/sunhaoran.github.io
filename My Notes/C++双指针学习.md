双指针法（Two Pointers）在数组操作中是一种高效且常用的技巧，尤其适用于需要原地修改数组并保持特定顺序的场景。通过维护两个指针（通常称为“快指针”和“慢指针”），可以在一次遍历中完成数组的筛选、去重或移除特定元素的操作，时间复杂度为 O(n)，空间复杂度为 O(1)

核心思想

快指针（fast）：负责遍历数组，检查每个元素是否满足条件（如不等于目标值、不重复等）
慢指针（slow）：负责记录满足条件的元素的位置，最终指向新数组的末尾

应用场景分析

移除元素（力扣 第 27 题）

目标：移除数组中所有等于 val 的元素，返回新长度 k
逻辑：快指针遍历数组，若当前元素不等于 val，则将其赋值给慢指针位置，并移动慢指针
关键点：慢指针始终指向新数组的下一个位置，最终返回慢指针的值即为新长度

删除有序数组中的重复项（ 力扣 第 26 题）

目标：删除重复元素，使每个元素只出现一次，返回新长度 k
逻辑：快指针遍历数组，若当前元素与前一个元素不同，则将其赋值给慢指针位置，并移动慢指针
关键点：利用数组有序的特性，只需比较当前元素与前一个元素是否相同

代码演示
```cpp
    class Solution {
    public:
        int removeDuplicates(vector<int>& nums) {
            int k = 1;
            for (int i = 1; i < nums.size(); i++) {
                if (nums[i] != nums[k - 1]) {
                    nums[k] = nums[i];
                    k++;
                }
            }
            return k;
        }
    };
```

删除有序数组中的重复项 II（力扣 第 80 题）

目标：删除重复元素，使每个元素最多出现两次，返回新长度
逻辑：快指针遍历数组，若当前元素与慢指针前两个元素不同，则将其赋值给慢指针位置，并移动慢指针
关键点：需要额外判断当前元素是否已经出现过两次，因此慢指针需要向前看两个位置

总结

双指针法在数组操作中的核心优势是高效和原地修改。通过合理设计快慢指针的移动规则，可以轻松解决移除元素、去重等问题，在实际应用中，需根据具体问题调整指针的移动条件，确保满足题目要求

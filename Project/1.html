项目背景

在传统的农业生产中，农户往往面临病虫害识别难、种植技术咨询渠道少的问题。 本项目通过整合 Coze (扣子) 强大的工作流编排能力与专属农业知识库， 构建了一个轻量级的小程序问答助手，旨在降低农业技术获取的门槛。

核心功能

📚 专属农业知识库

我们在 Coze 中整理了大量专业的农业文档、种植手册和病虫害图谱。

AI 不再是泛泛而谈，而是基于我们上传的本地知识库进行精准回答，确保建议的科学性和实用性。

💬 智能问答体验

用户只需在小程序中输入遇到的问题（如“水稻叶子发黄怎么办？”），AI 即可即时响应。

支持多轮对话，能够像真人专家一样追问细节，提供针对性的解决方案。

🚀 快速部署与集成

利用 Coze API 快速对接微信小程序后端。

实现了从用户提问、云端处理到结果返回的毫秒级响应，无需复杂的服务器维护。
